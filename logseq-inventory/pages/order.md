- order - ключевой шаг в pipeline после [[basket]]
- Связан с → [[basket]], [[payment]], [[shipment]]
- связан с [[product]] через [[basket]]
- order → shipment (или статус в order)
- order → payment (или метод/факт оплаты)
- Атрибуты::
	- id
	- user_id / buyer_id
	- basket_id
	- created_at
